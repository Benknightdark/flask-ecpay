詳細細節寫於：
[【Flask 教學】Python 綠界金流 API 信用卡串接](https://www.maxlist.xyz/2020/02/14/python-ecpay/)
* 撰寫時間為 2020-2-14，綠界 API 文件版本 V 5.1.38
* 資料庫使用 flask_sqlalchemy – ORM
* 語言 Python – flask
